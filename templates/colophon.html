Colophon

This website is created using open standards technology and free software. For those interested in details, the site is cobbled together using:

  • RDF for flexible and web-friendly data modeling
  • RDFa for embedded metadata; most content here has it
  • Dublin Core, FOAF, and the Bibliographic Ontology for data encoding
  • Python, including libraries like:
    • RDFLib for basic RDF storage and manipulation
    • RDFAlchemy for RDF-Object mapping to make it easier to display
    • web.py to map URLs to objects to display templates
  • Git for managing the files
  • XHTML and CSS
  • emacs for coding and writing
